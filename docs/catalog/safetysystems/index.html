Petzl выпускает спортивные страховочные системы для скалолазания и альпинизма, для спелеологии и каньонинга, для веревочных парков и для проката на скалодроме. В ассортименте бренда есть универсальные модели, есть беседки для мужчин, для женщин и для детей, есть нижние и верхние обвязки.

Спортивные обвязки можно разделить на два типа – с регулируемыми ножными обхватами и с нерегулируемыми. Например, женская обвязка LUNA с регулируемыми ногами отлично подойдет тем, кто любит универсальность, ее можно надеть как на шорты, так и на зимние утепленные штаны. Беседка SELENA практически такая же, но без регулировки ножных обхватов. Хотя самые популярные обвязки, конечно, это универсальные CORAX.

Альпинистские модели также можно разделить по количеству петель снаряжения, а специализированные обвязки для скитура можно надеть, стоя ногами на земле, не снимая лыж или кошек. Также представлен широкий ассортимент страховочных систем для веревочных парков, каньонинга, спелеологии, то есть практически любых видов высотных активностей. Комплект CORAX KIT подойдет тем, кто хочет сразу приобрести готовый набор для скалодрома.
